# 🤖 AI Personal Assistant

An AI-powered web assistant built using **Flask** and **Groq's Llama 3.3** language model. The application allows users to interact with an AI assistant for general queries and summarize lengthy emails through a simple and responsive web interface.

---

## 🚀 Features

- 💬 AI-powered question answering
- 📧 Email summarization
- ⚡ Fast responses using Groq API
- 🌐 Interactive web interface
- 🔒 Environment variable support for API security

---

## 📂 Project Workflow

1. User enters a question or email.
2. Flask receives the request.
3. Request is sent to the Groq API.
4. Llama 3.3 processes the prompt.
5. AI-generated response is returned.
6. Response is displayed on the web interface.
